# Getting Started

`v-gsap` is a batteries-included module for easy access to the power of GSAP

## Auto install

```bash [npx]
npx nuxi module add v-gsap
```

## Install manually

1. Install from npm

::code-group

```bash [npm]
npm i v-gsap
```

::

2. Add module to your `nuxt.config.ts`

::code-group

```ts [nuxt.config.ts]
modules: ["v-gsap"];
```

::
