# Getting Started

The Module can be easily installed in seconds.


## Install with nuxi


```bash [npx]
npx nuxi module add v-gsap-nuxt
```

## Install manually

1. Install from npm

::code-group

  ```bash [npm]
  npm i v-gsap-nuxt
  ```
::

2. Add module to your `nuxt.config.ts`

::code-group

  ```ts [nuxt.config.ts]
  modules: [
    'v-gsap-nuxt'
  ]
  ```
::
