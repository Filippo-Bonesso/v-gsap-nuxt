# Getting Started

`v-gsap-nuxt` is a batteries-included module for easy access to the power of GSAP


## Auto install


```bash [npx]
npx nuxi module add v-gsap-nuxt
```


## Install manually

1. Install from npm

::code-group

  ```bash [npm]
  npm i v-gsap-nuxt
  ```
::

2. Add module to your `nuxt.config.ts`

::code-group

  ```ts [nuxt.config.ts]
  modules: [
    'v-gsap-nuxt'
  ]
  ```
::
